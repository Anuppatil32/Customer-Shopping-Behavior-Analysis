📊 Customer Behavior Dashboard (Power BI + MySQL Project)

🚀 Project Overview
This project presents a Customer Behavior Dashboard built using Power BI and MySQL to analyze customer purchasing patterns, preferences, and trends.
The data was first analyzed using MySQL queries to extract meaningful insights and then visualized using Power BI for interactive reporting.

🎯 Key Features
📌 Interactive dashboard with slicers and filters
🗄️ Data analysis using MySQL queries
📍 Location and payment method-based insights
👥 Customer segmentation (Age, Gender, Category)
💰 Revenue and sales analysis
⭐ Customer satisfaction (Ratings)
📦 Seasonal purchasing trends

📈 Dashboard Insights
🔢 KPIs
Total Customers: 3.9K
Average Purchase Amount: $59.76
Average Review Rating: 3.75
🛍️ Category Insights
Clothing generates the highest revenue and sales
Accessories and Footwear follow
Outerwear has the lowest contribution
👥 Customer Insights
73% Non-Subscribers, 27% Subscribers
Young adults are the highest contributing segment
🌦️ Seasonal Insights
Sales are evenly distributed across seasons
Slight peak in Spring & Summer

Tools & Technologies
MySQL – Data querying & analysis
Power BI – Data visualization
CSV Dataset – Data source
Power Query – Data cleaning
